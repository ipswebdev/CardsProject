# RecruiterFlowProject

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 10.2.0.

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.

-Once the Project is successfully built, the app will redirect to /cards route where the cards will be fetched using the placeholder api
-The cards would be displayed according to the viewport size.
-To remove the card, click on the X in the top left corner of each card and it will be removed.
-To add a new card, simply click on the add card button, and a new card would be added to the list with the same title and image.
